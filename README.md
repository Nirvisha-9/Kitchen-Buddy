# Kitchen-Buddy

**Kitchen-Buddy** is a web application that helps users discover recipes based on the ingredients they already have at home. It simplifies meal planning by suggesting recipes you can make without additional shopping.  

## Features
- **Ingredient-based recipe suggestions:** Enter the ingredients available in your kitchen and get a list of recipes you can make.  
- **Recipe search:** Type in specific ingredients or dishes, and get recipes that match your inputs.  
- **User-friendly interface:** Easy to input ingredients and view recipe suggestions.  
- **Smart recommendations:** Maximizes the use of available ingredients to minimize waste.
